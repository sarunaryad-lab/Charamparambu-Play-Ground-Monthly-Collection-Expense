One-time Setup

Connect to Firebase so all group members share live data

1Create a free Firebase project

Go to console.firebase.google.com → Add project → name it charamparambu → Continue → Disable Analytics → Create project

2Create a Firestore database

In your project → Build → Firestore Database → Create database → Start in test mode → Choose a region near you (e.g. asia-south1) → Enable

3Register a Web App

Project settings (gear icon) → Your apps → Web (</>) → App nickname: cp-app → Register app → Copy the firebaseConfig values below

4Paste your Firebase config

Find these values in firebaseConfig object on Firebase console:

This is a one-time setup. Share the configured file with your group — everyone will connect to the same database automatically.
